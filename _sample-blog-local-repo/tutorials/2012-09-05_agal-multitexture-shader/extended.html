As always the most difficult are the initial steps. To get started I advise to read an article from Adobe explaining what the AGAL is, what are Vertex and Fragment programs, registers, available op-codes and AGALMiniAssembler: http://www.adobe.com/devnet/flashplayer/articles/what-is-agal.html. This article is not comprehensive enough and leaves you with many questions and unfortunately I couldn't find any additional information on Adobe websites explaining AGAL in greater detail. Anyway it is good enough to get you started. Additional handy information about registers could be found here: http://www.saltgames.com/2011/agal-registers/

\r\n

So now when you know all the basics and want to try creating some shaders based on one of many of the GLSL examples available on internet, read a great article from Devon_O on his blog: http://blog.onebyonedesign.com/actionscript/learning-agal-with-agalmacroassembler-and-opengl-examples/. If you read it attentively, you shouldn't have any problems understanding it as it guides you step by step how to do it.

/*\r\n//GLSL: /////////////////////////////////////////////////////////\r\n \r\n    vec2 p = -1.0 + 2.0 * gl_FragCoord.xy / resolution.xy;\r\n    // a rotozoom\r\n    vec2 cst = vec2( cos(.5*time), sin(.5*time) );\r\n    mat2 rot = 0.5*cst.x*mat2(cst.x,-cst.y,cst.y,cst.x);\r\n    vec3 col1 = texture2D(tex0,rot*p).xyz;\r\n \r\n    // scroll\r\n    vec3 col2 = texture2D(tex1,0.5*p+sin(0.1*time)).xyz;\r\n \r\n    // blend layers\r\n    vec3 col = col2*col1;\r\n \r\n    gl_FragColor = vec4(col,1.0);\r\n    \r\n//CONSTANTS: ////////////////////////////////////////////////////\r\n  fc0 = [ 1, 1, 1, 1 ]\r\n  fc1 = [ .5 * mTime, .1 * mTime, 2, 0 ]\r\n  fc2 = [ sin(.5 * mTime), cos(.5 * mTime), sin(.1 * mTime), .5 ]\r\n \r\n/////////////////////////////////////////////////////////////////\r\n*/  \r\n \r\n \r\n//Step 1: ft1 = vec2 p = -1.0 + 2.0 * gl_FragCoord.xy / resolution.xy;\r\nmov ft1, v0\r\nmul ft1.xy, ft1.xy, fc1.z\r\ndiv ft1.xy, ft1.xy, fc0.xy\r\nsub ft1.xy, ft1.xy, fc0.x\r\n \r\n//Step2: ft2 = vec2 cst = vec2( cos(.5*time), sin(.5*time) );\r\nmov ft2.x, fc2.y\r\nmov ft2.y, fc2.x\r\n \r\n//Step3: ft3 = mat2 rot = 0.5*cst.x*mat2(cst.x,-cst.y,cst.y,cst.x);\r\nmov ft0.x, fc2.w\r\nmul ft0.x, ft0.x, ft2.x\r\n \r\nmov ft3.x, ft2.x\r\nmul ft3.x, ft3.x, ft0.x\r\n \r\nmov ft3.y, fc1.w\r\nsub ft3.y, ft3.y, ft2.y\r\nmul ft3.y, ft3.y, ft0.x\r\n \r\nmov ft3.z, ft2.y\r\nmul ft3.z, ft3.z, ft0.x\r\n \r\nmov ft3.w, ft2.x\r\nmul ft3.w, ft3.w, ft0.x\r\n \r\n//Step4: ft4 = rot * p\r\nmul ft0.x, ft3.x, ft1.x\r\nmul ft0.y, ft3.y, ft1.y\r\nmul ft0.z, ft3.z, ft1.x\r\nmul ft0.w, ft3.w, ft1.y\r\nadd ft4.x, ft0.x, ft0.y\r\nadd ft4.y, ft0.z, ft0.w\r\nmov ft4.zw, fc0.zw\r\n \r\n//Step5: ft5 = vec3 col1 = ft0, ft0, fc2.z\r\ntex ft6, ft0, fs1<2d, repeat, linear, nomip>\r\n \r\n//Step7 ft7 = vec3 col = col2*col1;\r\nmul ft7, ft6, ft5\r\n \r\n//Step8 gl_FragColor = vec4(col,1.0);\r\nmov oc, ft7
